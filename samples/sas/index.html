Data mining in SAS

Target was to classify a binary target variable ("which KM0 vehicles have an higher discount than average?") understanding key variables in SAS.

Dataset was generated using scraping from autoscout.it (same scaper as in this other project of mine), preprocessed using Openrefine and Pandas. In SAS, we have used a bunch of models, ranging from decision trees to neural nets, using ROC/AUC as model picking on a holdout test and scored results.

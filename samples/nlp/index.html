Text mining

Target was to project generate a word embedding using gensim FastText library, and use it to train a LSTM model from KERAS to recommend the top 3 keywords that match the abstract content of "machine learning" papers related of the last 10 years published on the Scopus website.

Dataset was generated using scraping from scopus.com, preprocessed using spark in databricks. Modelling carried out in Colab.
This is a workgroup carried out during my masters degree.
